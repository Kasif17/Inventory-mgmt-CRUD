
# Product & Category


# Getting started 

## Requirement

* Node Js 
* MySql  

## Installation 




Clone the repository 

    git clone repo link

Install all the dependencies using npm 

    npm install 


Create MySQL database. name : "product-category" 


Start the local development server 

    nodemon app.js 


----------

**TL;DR command list**

    git clone Repo Link
    cd Product-Category
    npm install
    nodemon app.js
